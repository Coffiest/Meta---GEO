# テーブルデザインの差し込み方法

`felt.png` という1ファイルをこのフォルダに置くだけで、ポーカーテーブルの背景(フェルト部分)が
自動的にその画像に切り替わります。コードの変更は不要です。画像が無い場合、または読み込みに
失敗した場合は現行のグラデーション背景に自動でフォールバックします。

## 仕様

- ファイル名: `felt.png`(固定)
- 表示領域の縦横比: 概ね 5:6(幅よりやや高い楕円形)。`object-cover` で表示するため、
  枠に対して余裕を持ったデザインだと切り抜かれても破綻しにくくなります
- 角は `border-radius: 46%` で丸められた楕円形マスクがかかります(画像自体を楕円形に
  切り抜いておく必要はありません)
- 推奨解像度: 800×960px 程度以上

現状(スマホ縦画面・6人掛け・オポーネント5人が上部にアーチ状、自分は下部中央)の座席レイアウトは
`apps/web/src/components/PokerTable.tsx` の `SEAT_LAYOUT` で調整できます。テーブルデザインの
縦横比が大きく変わる場合はここも合わせて調整してください。
